# 🧠 Sistema Inteligente de Análise de Imagens com IA

Projeto desenvolvido para aplicar conceitos de **processamento de imagens**, **visão computacional** e **inteligência artificial** utilizando Python.

---

## 🎯 Objetivo

Criar um sistema capaz de:

* Processar imagens
* Aplicar filtros
* Analisar cores
* Gerar histogramas
* Realizar segmentação
* Detectar objetos com IA (YOLO)
* Interpretar resultados automaticamente

---

## 🛠️ Tecnologias utilizadas

* Python
* OpenCV
* NumPy
* Matplotlib
* Ultralytics YOLO (IA)

---

## ⚙️ Funcionalidades

### 🔹 Processamento de imagem

* Conversão para escala de cinza
* Aplicação de blur
* Detecção de bordas

### 🔹 Análise de cores

* Conversão para HSV
* Exibição dos canais (H, S, V)

### 🔹 Histograma

* Geração de gráfico de intensidade
* Interpretação da iluminação (clara ou escura)

### 🔹 Binarização

* Aplicação de threshold
* Segmentação da imagem

### 🔹 Inteligência Artificial

* Detecção de objetos com YOLO
* Identificação e contagem de objetos

---

## 📊 Pipeline do sistema

1. Aquisição da imagem
2. Pré-processamento
3. Análise de cores (HSV)
4. Geração de histograma
5. Binarização
6. Detecção de objetos (YOLO)
7. Exibição dos resultados

---

## 🖼️ Exemplo de saída

O sistema exibe:

* Imagem original
* Imagens processadas
* Histograma
* Detecção de objetos com IA

## 🚀 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/analise-imagem-ia.git
```

2. Acesse a pasta:

```bash
cd analise-imagem-ia
```

3. Crie um ambiente virtual:

```bash
python -m venv venv
```

4. Ative o ambiente:

```bash
venv\Scripts\activate
```

5. Instale as dependências:

```bash
pip install opencv-python numpy matplotlib ultralytics
```

6. Execute o projeto:

```bash
python atv.py
```

---

## 📌 Observações

* O modelo YOLO será baixado automaticamente na primeira execução
* Certifique-se de que a imagem está no caminho correto
